Instrucciones para correr el sistema:

1. En primer lugar, abrir una terminal en la ubicación en donde se encuentra el 
archivo "manage.py".
2. Ejecutar el comando:
	>> python manage.py runserver
3. Ir al navegador y poner lo siguiente: 
	http://localhost:8000/menu
4. Dentro de esa página se pueden crear y editar los menús.
